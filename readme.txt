Enable payments with Aur app in WooCommerce

== Description ==

Allow customers to use Aur app to purchase products in WooCommerce by simply using their phone number and accepting payment within the Aur app

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/plugin-name` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the WooCommerse -> Settings-> Payments -> Aur screen to configure the plugin
4. Input webkey and token from Aur to active payments

Requires WooCommerce to be installed.


== Frequently Asked Questions ==
